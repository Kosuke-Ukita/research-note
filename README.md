# ResearchNote

研究に関する日々の気づきや気持ちを綴る研究ノートです。

## 概要

このリポジトリは、研究活動を日記形式で記録し、目標のマイルストーンを設定して進捗を管理するためのものです。論文読みや日々の学び、課題などを体系的に記録し、研究の振り返りに活用します。

## ディレクトリ構造

```
ResearchNote/
├── README.md (このファイル)
├── diary/ (研究日記)
│   ├── README.md
│   ├── template.md
│   └── YYYY-MM-DD.md
├── papers/ (論文読みノート)
│   ├── README.md
│   ├── template.md
│   └── 著者_年_キーワード.md
└── milestones/ (目標とマイルストーン)
    └── README.md
```

## 使い方

### 1. 研究日記を書く

`diary/` ディレクトリで日々の研究活動を記録します。

```bash
# テンプレートをコピーして新しい日記を作成
cp diary/template.md diary/2026-02-15.md
```

日記には以下を記録します：
- 実施したこと
- 学んだこと
- 課題・疑問
- マイルストーンへの進捗
- 次のアクション

### 2. 論文読みノートを取る

`papers/` ディレクトリで読んだ論文の要約とメモを記録します。

```bash
# テンプレートをコピーして新しい論文ノートを作成
cp papers/template.md papers/Smith_2024_DeepLearning.md
```

論文ノートには以下を記録します：
- 論文の基本情報
- 概要と主な結果
- 重要なポイント
- 自分の研究への応用

### 3. マイルストーンを管理する

`milestones/README.md` で研究目標を設定し、進捗を追跡します。

- 短期目標（1-3ヶ月）
- 中期目標（3-6ヶ月）
- 長期目標（6ヶ月以上）

達成したマイルストーンは「達成したマイルストーン」セクションに移動します。

## 運用のヒント

- **定期的な記録**: 毎日または週に数回、日記を書く習慣をつける
- **振り返り**: 週次や月次で過去の日記を読み返し、進捗を確認
- **マイルストーンの見直し**: 定期的に目標を見直し、必要に応じて調整
- **論文の整理**: 読んだ論文は必ずノートを取り、後で参照できるようにする

## 目的

このノートは**個人の記録用**であり、以下の目的で使用します：

- 研究活動の可視化
- モチベーションの維持
- 学びの定着
- 進捗の追跡
- 振り返りの材料
