# 論文読みノート

読んだ論文の要約、メモ、自分の研究への応用などを記録します。

## ファイル命名規則

`著者名_年_キーワード.md` または `会議名_年_キーワード.md` の形式で作成してください。

例: 
- `Smith_2024_DeepLearning.md`
- `NeurIPS_2024_Attention.md`

## 使い方

1. `template.md` をコピーして新しい論文ノートを作成します
2. 論文の情報とメモを記入します
3. 自分の研究への応用可能性を考えます

## ディレクトリ構造

```
papers/
├── README.md (このファイル)
├── template.md (テンプレート)
└── ... (論文ごとのノート)
```
